Angular UI: Challenge

Challenge!! The Sushi Shop

จากที่เราได้ทำการศึกษา UI Router ไป เราจะมาลองสร้างด้วยตัวเองดู

ให้ทำการสร้างร้านซูชิ มี 2 หน้าเว็บ คือ menu และ checkout

menu จะแยกออกเป็น 3 ประเภท คือ sushi appetizer drink

แต่ละประเภทจะใช้ template เดียวกัน

โดยกรณีที่ลิงค์ไม่ถูกหน้าให้ทำการ redirect มาที่หน้าเมนูซูชิ

Check Point 1: Create Menu state

ให้ทำการสร้าง state menu เป็น abstract state โดยมีจะเรียกใช้ template ชื่อไฟล์ menu.html

menu.html

<div class="col-md-8" ui-view="menu">
	<ul class="nav nav-pills navbar-default">
		<li><a ui-sref="#">Sushi</a></li>
		<li><a ui-sref="#">Appetizer</a></li>
		<li><a ui-sref="#">Drink</a></li>
	</ul><div ui-view="menulist"></div></div>
	<div class="col-md-4" ui-view="order">
	</div>
</div>

โดยใน controller ของ menu มีรายละเอียดดังนี้

function ($scope, $state) {
	$scope.orderItem = [];
	$scope.increase = function (item) {
		if (item.quantity !== null) {
			item.quantity += 1;
		}
	};
	$scope.decrease = function (item) {
		if (item.quantity !== null && item.quantity > 0) {
			item.quantity -= 1;
		}
	};
	$scope.orderCall = function (item) {
		if (item.quantity !== null && item.quantity > 0) {
			var addItem = angular.copy(item);
			$scope.orderItem.push(addItem);
			item.quantity = 0;
		}
	};
	$scope.removeOrder = function (item) {
		var index = $scope.orderItem.indexOf(item);
		$scope.orderItem.splice(index, 1);
	}
	$scope.checkOut = function () {
		var params = {"orderItem":$scope.orderItem};
		console.log(params);
		$state.go("checkout", params);
	};
}

Check Point 2: Create State (Sushi, Appetizer, Drink)

สร้าง state ขึ้นมาใหม่ 3 state ได้แก่ sushi, appetizer, drink

โดยให้สืบทอดมาจาก menu โดยจะเรียกใช้ 2 views

  • menulist ใช้ templateUrl เป็นไฟล์ menu.html
  • order ใช้ templateUrl เป็นไฟล์ order.html

และกำหนดถ้าหา state ไม่เจอให้ไปยัง state sushi

ไฟล์ menu.html

<div class="list-group">
	<div class="list-group-item row" ng-repeat="item in menuItem" ng-init="item.quantity = 0">
		<h2 class="col-md-4">{{item.name}}</h2>
		<div class="col-md-2"><em>{{item.price | number:2}} Yen </div>
		<div class="col-md-2"><p>X {{item.quantity}}</p> <button class="btn btn-default" ng-click="increase(item)">increse</button>
		<button class="btn btn-default" ng-click="decrease(item)">decrese</button></div>
		<div class="col-md-2"><button class="btn btn-default pull-right" ng-click="orderCall(item)">+</button></div>
	</div>
</div>

ไฟล์ order.html

<div class="list-group">
	<div class="list-group-item row" ng-repeat="item in orderItem">
		<h2 class="col-md-4">{{item.name}}</h2>
		<div class="col-md-4"><em>{{item.price | number:2}} Yen </div>
		<div class="col-md-2"><p>X {{item.quantity}}</p></div>
		<div class="col-md-2"><button class="btn btn-default pull-right" ng-click="removeOrder(item)">X</button></div>
	</div>
	<button ng-show="orderItem.length > 0" ng-click="checkOut()" class="btn btn-danger col-md-12">Check Out</button>
</div>

Answer of Check Point 2

angular.module('application', ['ui.router'])
    .config(function ($stateProvider, $urlRouterProvider) {
        $urlRouterProvider.otherwise('/sushi');

Check Point 3: Resolve Data

ให้ทำการเพิ่มข้อมูลลงไปในแต่ละ state โดยมีรายละเอียดดังนี้

  • sushi state
    • {{item.name}} ราคา {{item.price}} yen
  • appetizer state
    • {{item.name}} ราคา {{item.price}} yen
  • drink state
    • {{item.name}} ราคา {{item.price}} yen

โดยใน controller ของ views menulist จะมีการเรียกใช้ดังนี้

// for menulist view in sushi state
controller: function ($scope, sushiItem) {
	$scope.menuItem = sushiItem;
}

Check Point 4: Create Check Out State

สร้าง checkout state โดยการที่เราจะมายัง state นี้ได้จะเรียกผ่านฟังก์ชั่น $state.go()

และกำหนดให้ checkOut() ใช้งาน $state.go() ส่งตัวแปร orderItem มาด้วย

checkOut() ใน menu state

.state('menu', {
	...
	controller: function ($scope, $state) {
		$scope.orderItem = [];
		$scope.increase = function (item) { ... };
		$scope.decrease = function (item) { ... };
		$scope.orderCall = function (item) {... };
		$scope.removeOrder = function (item) {... };
		$scope.checkOut = function () { 
			// get all order and check out
			var params = {"orderItem":$scope.orderItem};
			// ???? send params to state checkout
		};
	}
})

โดย checkout state จะใช้ templateUrl เป็น checkout.html

ไฟล์ checkout.html

<div>
	<ul class="nav nav-pills navbar-default">
		<li><a ui-sref="menu.sushi">Back</a></li>
	</ul>
</div>
<div class="list-group">
	<div class="list-group-item row" ng-repeat="item in orderItem">
		<h2 class="col-md-4">{{item.name}}</h2>
		<div class="col-md-4">

		<h2 class="col-md-4">Total</h2>
		<h2 class="col-md-4">{{sum}} Yen</h2>
	</div>
	<button ng-click="payMoney()" class="btn btn-danger col-md-12">Pay</button>
</div>

และ checkout จะรับค่า orderItem ผ่านทาง params property โดยมี controller ดังนี้

 function ($scope, $stateParams, $state) {
	$scope.orderItem = $stateParams.orderItem;
	$scope.sum = 0;
	angular.forEach($scope.orderItem, function (item) {
		$scope.sum += (item.price * item.quantity);
	});
	$scope.payMoney = function () {
		alert("Thank you.");
		$state.go('menu.sushi');
	}
}

Answer of Check Point 4

angular.module('application', ['ui.router'])
    .config(function ($stateProvider, $urlRouterProvider) {
        $urlRouterProvider.otherwise('/sushi');
		$stateProvider
            .state('menu', { 
				... 
				$scope.checkOut = function () {
					var params = {"orderItem":$scope.orderItem};
					$state.go("checkout", params);
				};
			})
			.state('menu.sushi', { ... })
			.state('menu.appetizer', { ... })
			.state('menu.drink', { ... })
			.state('checkout', {
				url: '/checkout',
				params: {'orderItem':''},
				templateUrl: 'checkout.html',
				controller: function ($scope, $stateParams, $state) {
					$scope.orderItem = $stateParams.orderItem;
					$scope.sum = 0;

						$scope.sum += (item.price * item.quantity);
					});
					$scope.payMoney = function () {
						alert("Thank you.");
						$state.go('menu.sushi');
					}
				}
			});
    });
