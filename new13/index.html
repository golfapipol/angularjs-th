ก้าวข้าม V1.2 สู่ V1.3 from ng-learn

แนวคิด

AngularJS V. 1.3 จุดประสงค์หลักของเวอร์ชั่นนี้คือการเปลี่ยนแปลงส่วนหลักของ AngularJS Core ให้กลายเป็น Angular 2.0 ได้ง่ายขึ้น

Feature เด่นๆ ที่เพิ่มเข้ามา

เพิ่ม One-Way Binding

เพิ่ม ngModelOptions เข้ามาช่วยให้จัดการ ngModel ง่ายขึ้น

เพิ่ม Directive ngMessages เข้ามา

Breaking Changes ของที่พัง!

เลิก support IE8 แล้วจ้าาาา

input ที่เป็นตระกูล date, time, datetime-local, month, week ทั้งหลาย ข้อมูลต้องเป็น Date Object

$animate ต้องใส่ CSS Transition เพิ่มเองบางส่วน ไม่มี Default ให้แล้ว

$http interceptor เปลี่ยนวิธีการใช้งาน
